Über MovieWeb / About MovieWeb

Projektbeschreibung / Project Description

MovieWeb ist eine Webanwendung zur Verwaltung von Filmlisten. Benutzer können ihre Lieblingsfilme speichern, bewerten und kommentieren. MovieWeb is a web application for managing movie lists. Users can save, rate, and comment on their favorite movies.

Funktionen / Features

  • Benutzerverwaltung / User Management
    • Registrierung und Login / Registration and login
    • Persönliche Filmlisten / Personal movie lists
  • Filmverwaltung / Movie Management
    • Filme hinzufügen und bearbeiten / Add and edit movies
    • Bewertungssystem / Rating system
    • Kommentarfunktion / Comment feature
  • API-Integration / API Integration
    • OMDb API für Filminformationen / OMDb API for movie information

Technologien / Technologies

  • Backend / Backend
    • Python / Flask
    • SQLAlchemy / SQLite
    • RESTful API
  • Frontend / Frontend
    • HTML5 / CSS3
    • JavaScript
    • Bootstrap 5

Entwicklung / Development

MovieWeb wurde mit Fokus auf Benutzerfreundlichkeit und Robustheit entwickelt. Die Anwendung verwendet moderne Webtechnologien und folgt Best Practices für Sicherheit und Performance. MovieWeb was developed with a focus on user-friendliness and robustness. The application uses modern web technologies and follows best practices for security and performance.
